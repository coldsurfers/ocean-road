# ErrorUI

네트워크 에러, 알 수 없는 에러 등 에러 상태를 표시하는 복합 컴포넌트입니다. 재시도 버튼이 포함되어 있습니다.

## 사용법

```tsx
import { ErrorUI } from '@coldsurfers/ocean-road'

// 네트워크 에러
<ErrorUI.NetworkError onClickRetry={() => refetch()} />

// 알 수 없는 에러
<ErrorUI.UnknownError onClickRetry={() => refetch()} />
```

## 서브 컴포넌트

### ErrorUI.NetworkError

| prop | 타입 | 필수 | 설명 |
|------|------|------|------|
| `onClickRetry` | `() => void` | ✅ | 재시도 버튼 클릭 시 호출되는 콜백 |

### ErrorUI.UnknownError

| prop | 타입 | 필수 | 설명 |
|------|------|------|------|
| `onClickRetry` | `() => void` | ✅ | 재시도 버튼 클릭 시 호출되는 콜백 |
